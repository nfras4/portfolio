00 / identity
who this is for and why it exists

A brand for one person, written so a future hire can match it.

This guide documents the visual system already running on nickwfraser.dev. It isn't a redesign. It's a snapshot of decisions that earned their place across Tally, GoCard, Monkey Barrel, and the portfolio itself, written down so the next site, deck, or doc looks like the same person made it.

Nick Fraser. Brisbane. Finance + Business Analytics at UQ, graduating 2027. Building Tally (MBS billing copilot for paediatric orthopaedics), GoCard Insights (BSAN4204 capstone), Monkey Barrel (multiplayer arcade on Cloudflare Workers + DOs), and running Tek Monkeys as a registered IT business.

The brand's job is to make all of that read as one person, not five. Warm paper, restrained type, one rust accent, and a single repeated geometric motif: the lattice.

the math (don't change)
lift = sin(t · 1.4 − dist · 1.6) · 0.35 + max(0, 0.9 − dist) · 0.6. Hue ramps 0.06 → 0.14 with lift. This is the signature.

02 / color
six tones, one signature

Warm paper, warm ink, one rust.

OKLCH all the way down. Every value sits in the warm half of the wheel (hue 42 to 75). The rust accent is the only saturated color in the system. That scarcity is what makes it work.

paper 0.96 / 75 page bg
elev 0.93 / 70 tabs, nav-bg
kraft 0.89 / 70 cards, stages
dim 0.46 / 60 labels, keys
ink 0.21 / 50 body, headlines
★
rust 0.46 / 42 accent, single use

usage rules

  • Rust is the only saturated value. If you reach for green, blue, or yellow, stop. The system breaks.
  • Never put rust on rust. Rust always lives against paper, kraft, or ink. Soft-rust tint (--accent-soft, 14% alpha) is the one exception for fill states.
  • Body type is ink (0.21), not pure black. Labels and metadata drop to dim (0.46).
  • Use OKLCH literals, not hex. Hex hides what you're doing on the lightness axis.

05 / motion
one curve, three durations

One easing function, used everywhere.

Motion is the cheapest place to look sloppy. Pick one curve and one tempo, use them on every transition. The portfolio already does this; codifying the values keeps it that way.

Three things move on hover: row color shifts to rust, row gets 8px of left-padding (slide in), arrow translates (+2px, −2px). Same pattern on every link in the system.

entry stagger · fade-up sequence
0.05s status → 0.10s name → 0.18s pitch → 0.28s meta → 0.36s links

Hero page-load uses fadeUp (translateY 8px + opacity) on five elements with cascading animationDelay. Don't add more steps. Five is the budget. Status dot uses nav-pulse (2.4s loop, 0 to 6px box-shadow ring), the only persistent motion on the page.

06 / voice
how nick writes, so others can match

Direct sentences. Specific nouns. No edge.

do · specific over generic

"Paste a surgical report, get the Medicare item codes ready to claim."

Named the product action, named the user goal, no adjectives. Reads like a doctor would describe it.

don't · cliche stack-bragging

"Built on the edge with cutting-edge cloud-native technology."

"Edge" tells the reader nothing about the product. Name the actual differentiator instead (Workers + Durable Objects, $0/mo infra, single-tenant).

do · admit the unglamorous part

"Client-facing billing, scheduling, and the unglamorous parts of running a registered business."

Honesty signals competence. Don't hide the boring work; it's part of what differentiates a real practitioner from someone reciting a tech stack.

don't · em dashes

"A tool — for clinicians — that ships fast."

The brand uses commas, periods, and colons. Em dashes get over-used by writers trying to sound thoughtful. Strip them.
